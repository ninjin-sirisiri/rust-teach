# Project: Rust Teaching Website with Leptos

## 概要
Leptosを使用して、Rustの基礎から実践的な内容までを学べる学習サイトを構築します。

## 技術スタック
- **Framework**: Leptos (CSR with Trunk)
- **Language**: Rust
- **Styling**: Vanilla CSS (Modern, Premium Design)
- **Icons**: Lucide (via leptos-lucide if possible, or SVG)

## 開発ルール
- コンポーネントは可能な限りファイル分割する。
- 日本語で対応する。
- プレミアムなデザインを追求する。
- 開発サーバーの起動前にはユーザーの承認を得る。
